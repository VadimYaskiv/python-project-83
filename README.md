## Web page analyzer
### Descriptioin
The application has a UI and can be launched via web browser.
Check any URL to explore SEO-quality.

### Minimum requirements
1. Pip
2. Poetry
3. Python
4. Flask
5. Gunicorn
6. Dotenv
7. Psycopg2
8. BeautifulSoup
9. Postgresql

### Install
```bash
$ git clone https://github.com/VadimYaskiv/python-project-83.git
$ cd python-project-83
$ make install 
$ make build
```

### Run
```bash
$ make start
```
